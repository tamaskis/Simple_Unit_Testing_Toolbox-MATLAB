Installation

  1. Download the toolbox from File Exchange or https://github.com/tamaskis/Simple_Unit_Testing_Toolbox-MATLAB.
  2. Open the INSTALL folder.
  3. Double-click on Simple Unit Testing Toolbox.mltbx.

MATLAB will automatically perform the installation and add all the contents of the toolbox folder to the MATLAB search path.

Opening this documentation using the MATLAB Command Window.

To open the home page (this page) of the toolbox documentation in MATLAB, type the following in the Command Window:

doc_SUTT

To open the documentation of a specific function with name function_name, type the following in the Command Window:

doc_SUTT function_name

Creating a Test Suite

To create a test suite, we simply need to initialize a TestSuite object.

test_suite = TestSuite

test_suite = 

  TestSuite with properties:

        tests: {}
         name: ''
            N: 0
    terminate: 0

We can also give the test suite a name during initialization, and also specify whether or not we want the test suite to terminate at the first failed test. For more details, see the full TestSuite documentation below.

  • TestSuite Class defining a test suite.

Adding Tests to a Test Suite

Consider two near identical arrays.

X1 = [1  1;
      1  1];
X2 = X1+0.00001;

These arrays are equal to 4 decimal places, but not equal to 5 decimal places. We can add tests to test for this expected behavior.

test_suite.add_test(TestEqual(X1,X2,'equal to 4 decimal places',4));
test_suite.add_test(TestNotEqual(X1,X2,'not equal to 5 decimal places',5));

Running a Test Suite

To run the test suite, we can simply do

test_suite.run();

equal to 4 decimal places: Passed. (1/1)
not equal to 5 decimal places: Passed. (2/2)
